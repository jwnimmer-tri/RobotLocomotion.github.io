pydrake.systems.pyplot_visualizer¶

class pydrake.systems.pyplot_visualizer.PyPlotVisualizer(draw_period=0.03333333333333333, facecolor=[1, 1, 1], figsize=None, ax=None, show=None)¶

Bases: pydrake.systems.framework.LeafSystem_[float]

Base class from planar visualization that relies on pyplot.

In the configuration set up here, this visualizer provides one visualization window (self.fig) with axes (self.ax). The axes can optionally be supplied externally to allow other visualizers to overlay additional information.

Subclasses must: - During initialization, set up the figure bounds and register input port with the appropriate message type. - Override the draw method to parse the input and draw the robot in the appropriate state.
